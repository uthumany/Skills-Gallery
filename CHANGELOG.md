## [1.0.0] - 2026-06-27

### 🎉 Initial Release

The first public release of Hermes Skills Gallery — the ultimate all-in-one gallery of AI agent skills for Hermes Agent.

### Added

#### Core Features
- **CLI Tool** (`hsk` / `hermes-skills`) with rich terminal UI
- **Skill Listing** — Browse all available skills with `hsk list`
- **Full-Text Search** — Find skills by name, description, or tags with `hsk search`
- **Category Browsing** — Filter skills by category with `hsk browse --category`
- **One-Command Install** — Install skills directly with `hsk install <skill-name>`
- **Skill Details** — View metadata, dependencies, and ratings with `hsk info`
- **Gallery Stats** — Get gallery statistics with `hsk stats`
- **Validation** — Validate skill metadata with `hsk validate`

#### Content
- **1,600+ Skills** across 40+ categories
- **40+ Categories** including Development, DevOps, Data Science, Web, Security, and more
- **200+ Tags** for granular skill discovery
- Community-contributed skill catalog

#### Installation Methods
- PyPI package (`pip install hermes-skills-gallery`)
- npm package (`npm install -g hermes-skills-gallery`)
- yarn, pnpm, bun support
- npx one-off execution
- deno support
- curl install script

#### Documentation
- Comprehensive README with usage examples
- Contributing guide with skill submission workflow
- Code of Conduct
- Security policy
- Issue and PR templates

#### Infrastructure
- Python package build with setuptools
- CI-ready test suite structure
- `.gitignore` configuration
- GitHub community health files
